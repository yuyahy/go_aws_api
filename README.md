# go_aws_api
Repository for Go API on AWS

## Overview
AWS上にGoのAPIサーバーを立てるサンプルリポジトリ
API Gateway + Lambda + DynamoDBを使用している

## Requirements
go version go1.22.6 darwin/amd64
